# BC Fischer — Будівельна компанія

Сайт інвестиційно-будівельної компанії BC Fischer. Односторінковий додаток з двомовною підтримкою (UK/EN).

## Запуск

```bash
npm install
npm run dev
```

## Секції

- **Hero** — креативний дизайн з 3D краном, blueprint-сіткою та геометричними акцентами
- **Про компанію** — опис, фото команди, статистика
- **Послуги** — 12 видів будівельних робіт
- **Портфоліо** — реалізовані проекти з фотографіями
- **CTA** — заклик до дії
- **Контакти** — форма зворотного зв'язку та контактна інформація
- **Footer** — навігація, контакти, соцмережі

## Технології

- React 19
- Vite
- SCSS (BEM)
- Framer Motion
- Lucide React
- Google Material Symbols

## Структура

```
src/
  assets/          # Зображення (логотипи, кран)
  components/      # Компоненти секцій
  contexts/        # Мовний контекст (i18n)
  data/            # Конфігурація сайту
  locales/         # Переклади (uk.json, en.json)
  styles/          # SCSS (змінні, reset, main)
```
